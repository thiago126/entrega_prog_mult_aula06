O que é Ionic?

O Framework Ionic é um SDK de código livre que possibilita aos desenvolvedorea a criação de apps de alta qualidade com tecnologias web como HTML, CSS e JavaScript.

O que é Ionic Cloud?

A Ionic Cloud é uma plataforma na nuvem para gerenciamento e escalabilidade dos apps Ionic com serviços integrados como notificações push, builds nativos, autenticação de usuário e atualização ao vivo.
